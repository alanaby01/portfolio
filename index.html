A little about me

I'm a full-stack developer and data engineer currently at Optum, where I've spent the last three years wearing a few different hats across engineering.

I started out building and maintaining an internal infrastructure platform with .NET and React, then moved into leading a design system powering shared UI components across web and mobile with React Native. These days I'm on the Clinical Language Intelligence team, designing ETL pipelines across AWS, Azure, and Databricks — and building an agentic platform that lets users query clinical databases using plain natural language.

I like working at the intersection of frontend craft and data systems, the kind of problems that require thinking about both the experience and the engine underneath it.
